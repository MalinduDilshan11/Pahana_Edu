Billing System

Billing Workflow

Follow this step-by-step process to create accurate bills with automatic discounts.

1

Select Customer

Choose a customer ID from the dropdown. A valid customer is required.

2

Add Items

Select item names, enter quantities, and click "Add to Cart". Repeat for multiple items.

3

Review Discounts

Discounts are applied automatically: 5% (>2500), 10% (>5000), 15% (>10000).

4

Generate and Print

Click "Save Bill" to finalize, then "Print Bill" to view or print. Use "Cancel" to close the print preview.

Troubleshooting

Common Error Messages

"Customer not found"

Verify the customer ID exists in your database.

"Connection failed"

Check if backend server is running on port 8080.

"Invalid input format"

Ensure all required fields contain valid data.

Technical Requirements

Backend server: http://localhost:8080/PahanaEduService/resources/
Recommended browsers: Chrome, Firefox, Edge
JavaScript must be enabled
Stable internet connection required

Performance Tips

  • Clear browser cache if experiencing loading issues
  • Avoid opening multiple tabs of the application
  • Log out properly to prevent session conflicts

Security Best Practices

  • Always log out when finished
  • Don't share login credentials
  • Use private browsing for shared computers
  • Keep your browser updated
